### ■コンセプト
頭痛に負けない習慣作り\
＞開発するものは占い(相性占い)とする。\
＞作業時間は土曜日と社員旅行の日を除く朝8:30～9:30の1時間とする。\
 　＞日曜も入れるとしんどい、、、\
＞出社・起床できなかった場合は作業時間なしとする。\
＞最低限相性占いだけは完成させ、残りは占いの種類を増やしていく形とする。\
＞他の優先作業がある場合はそちらを優先する。
